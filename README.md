# H5魔方游戏

一个基于HTML5的3D魔方游戏，支持完整的魔方旋转操作、打乱、求解功能。

## 功能特性

- 🎯 **完整3D渲染**：使用CSS3 3D变换实现逼真的魔方效果
- 🎮 **多种控制方式**：
  - 鼠标拖拽旋转视角
  - 键盘快捷键（U/D/L/R/F/B）
  - 支持Shift+字母进行逆时针旋转
  - 触摸屏支持
- 🔄 **完整魔方操作**：
  - 6个面的顺时针/逆时针旋转
  - 打乱功能（随机旋转）
  - 自动求解（回溯移动历史）
- 📱 **响应式设计**：适配各种屏幕尺寸
- 🧪 **完整的测试套件**：验证魔方逻辑的正确性

## 快速开始

1. 打开 `index.html` 文件即可开始游戏
2. 使用鼠标拖拽旋转魔方视角
3. 使用按钮或键盘快捷键旋转魔方各面
4. 点击"打乱"随机打乱魔方
5. 点击"求解"自动恢复魔方

## 控制说明

### 键盘控制
- `U`/`U` : 上层面 顺时针/逆时针
- `D`/`D` : 下层面 顺时针/逆时针  
- `L`/`L` : 左面  clockwise/逆时针
- `R`/`R` : 右面  clockwise/逆时针
- `F`/`F` : 前面  clockwise/逆时针
- `B`/`B` : 后面  clockwise/逆时针
- **注意**: 按Shift+字母进行逆时针旋转

### 鼠标控制
- 拖拽魔方：旋转视角
- 点击按钮：旋转对应面

## 技术实现

### 魔方表示
魔方状态使用对象结构表示，每个面包含9个色块：
```javascript
{
    U: ['white', 'white', ...],  // 上面
    D: ['yellow', 'yellow', ...], // 下面
    F: ['green', 'green', ...],   // 前面
    B: ['blue', 'blue', ...],     // 后面
    R: ['red', 'red', ...],       // 右面
    L: ['orange', 'orange', ...]  // 左面
}
```

### 旋转算法
使用数学映射实现魔方各面的旋转，确保：
- 面内色块正确旋转
- 相邻面的边缘色块正确交换
- 支持顺时针和逆时针旋转

### 3D渲染
- 使用CSS `transform-style: preserve-3d`
- 27个小方块组成3x3x3结构
- 每个小方块有6个面，根据位置显示对应颜色

## 开发说明

### 魔方核心类
`RubiksCube` 类提供所有魔方操作：
- `rotateFace(face, clockwise)` : 旋转指定面
- `shuffle(numMoves)` : 随机打乱
- `reset()` : 重置到解决状态
- `isSolved()` : 检查是否已解决
- `getState()` / `importState()` : 状态序列化

### 控制器类
`CubeController` 管理游戏：
- 3D渲染和场景管理
- 用户输入处理
- 游戏状态管理（步数、计时）
- 魔方操作触发
